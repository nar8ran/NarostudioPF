BGM 配信向け 明るい・優しい

潮に咲く

雑談配信や導入シーンを想定した、やわらかく親しみのある空気感のBGM。和風と自然モチーフを穏やかに混ぜ、長時間再生しても疲れにくい質感を意識しました。

  • 用途：配信・動画・待機画面向けBGM
  • 担当：作曲 / 編曲 / ミックス
  • 工夫：ループ耐性、声とぶつからない帯域設計、印象に残る導入
ジングル 短尺 キャッチー

モズとフルート

数秒で雰囲気を切り替える用途を想定した短尺ジングル。かわいさと幻想感を両立し、配信の転換点やコーナー開始の合図として機能する設計を意識しました。

  • 用途：配信コーナー切替 / 動画トランジション
  • 担当：作曲 / アレンジ
  • 工夫：短時間での印象形成、アイキャッチ性、余韻処理
楽曲 ポップ キャラクター性

campana

キャラクターの明るさや前向きさを押し出すことを想定したポップ楽曲。メロディの親しみやすさを優先しつつ、展開で飽きが出ないようセクションごとに質感変化を入れています。

  • 用途：活動テーマ曲 / MV / 配信用オリジナル曲
  • 担当：作曲 / 編曲 / ミックス
  • 工夫：歌いやすい音域設計、覚えやすいサビ、展開差

制作プロセス

実技試験や面接で話しやすくするため、作り方を構造化して見せる例です。

1

要件整理

用途、尺、ターゲット、感情、参考作品、配信か映像かなどを整理して方向性を定めます。

2

モチーフ設計

旋律、コード感、楽器編成を先に決め、世界観に対して必要な音の役割を切り分けます。

3

アレンジ・調整

ループ性、声との共存、印象の持続を確認しながら編曲と質感設計を進めます。

4

納品視点で仕上げ

尺調整、書き出し形式、用途別バリエーションなど、実務で使いやすい形を意識してまとめます。

プロフィール / 採用向け補足

音源だけでは伝わりにくい「人となり」と「仕事の進め方」を添える例です。

自己紹介

デザイン系の学びを背景に、音だけでなく見せ方・世界観・受け手の印象まで含めて設計することを意識しています。 明るく可愛い質感の楽曲やBGMを得意としつつ、和風・幻想的・物語調の表現にも対応可能です。

面接で話せる要素

  • 依頼意図をどう読み取り、音に落とし込んだか
  • 短尺・長尺で設計をどう変えているか
  • 声や映像と共存させるための工夫
  • 修正対応ややり取りで意識している点
